# Style Vitality

The article visualization style should inherit the lively parts of the Xiaohei illustration skill. Native handwritten Chinese labels are part of the image by default.

Use:

- Pure white background and sparse black hand-drawn line art.
- Slightly wobbly pen lines, not polished vector geometry.
- Clean low-tech physical metaphors: switches, levers, carts, gears, funnels, workbenches, meters, labels, scraps, parcels, and small machines.
- A small square worker character doing the core conceptual action with a deadpan serious expression.
- Strange but clean object logic: the visual should feel like an absurd product sketch, not a formal classroom diagram.
- Large whitespace, with the main subject around 40%-60% of the canvas.
- Light accent colors only when they clarify action: orange for paths/arrows, red for warnings or friction, blue for secondary state, green/yellow for object separation.
- Short handwritten Chinese annotations directly generated in the image. Use at most 5-8 labels, preferably 2-8 characters each.

Do not use:

- Readable Chinese text in textless prompts.
- Cute mascot posing, idle corner decoration, or poster-like character treatment.
- PPT slide composition, corporate vector infographics, realistic UI, dense charts, or title blocks.
- Decorative background texture, gradients, shadows, or polished commercial illustration style.

Contract:

In native text mode, Chinese labels appear in the visual prompt and are rendered by the image model. Use legacy overlay mode only when the user explicitly wants deterministic text safety.
