# Visual System

Use 16:9 horizontal article illustrations with mostly white background, black hand-drawn line art, low-tech physical metaphors, lots of whitespace, and light accent colors only when needed.

The visual should have Xiaohei-style vitality: slightly wobbly pen lines, a strange but clean physical object logic, and a small square worker character doing the core conceptual action with a deadpan serious expression.

Prefer concrete low-tech mechanisms over abstract shapes: switches, levers, carts, gears, funnels, workbenches, meters, labels, scraps, parcels, and small machines.

The result is not PPT, not infographic, not UI, not a dense chart, and not vector corporate art.

In native text mode, render a few short Chinese handwritten labels directly in the image. The text should feel like part of the sketch, not a typeset layer. Use legacy blank placeholders only for deterministic overlay mode.
