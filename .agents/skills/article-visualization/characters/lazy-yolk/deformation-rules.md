# Deformation Rules

Deformation is disabled by default and only appears when the manifest mode, `structure_type`, and `anchor_type` match an allowed rule.

- `melting-funnel`: gently sags into a funnel-like body for filtering concept metaphors.
- `wedged-yolk`: lightly wedged in a bottleneck mechanism.
- `flattened-yolk`: mildly flattened by pressure while staying recognizable.
